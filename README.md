# Perceptron
+ Implimentation of perceptron to classify against And , OR Data
+ Modularized Coding
+ Logging
+ Progress bar
+ Docstring
+ Saving Models
+ Saving Plots

## How to use 
+ git clone [copy the link]
+ Go to Folder run the clone command
```bash
  cd perceptron
```
+ To install dependencies
```bash
  pip install -r requirements.txt
```
+ Run OR Model
```bash
  python or.py
```
+ Run AND Model
```bash
  python and.py
```
